На сайте docs.djangoproject.com/en/4.2/ref/templates/builtins/#url будет пример использования тега

{% url 'some-url-name' v1 v2 %}
v1 и v2 это параметры, которые могут отсутствовать

Визуально ничего не изменится, но если открыть код страницы, то можно будет увидеть разницу

render = функция загрузки шаблона, его обработки и возвращения результата в виде объекта HTTP-ответа
reverse = функция формирования URL-адресса на основе имени маршрута или представления
redirect = функция перенаправления по указаному URL-адрессу

Варианты формирования URL-маршрутов в шаблоне
{% url 'a' a a %}
{% url 'a' a=a a=2 %}
{% url 'a' %}
{% url 'a' 'a' 'a' %}
